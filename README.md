Asteroids game in python with
-  auto shoot
-  scoring system (10 pts per hit, 5 more pts on small asteroids)
-  stage system (move up stage every 200 pts)
-  dynamic difficuly (asteroids spawn's frequency increases at every stage but so does the player fire rate)
-  at stage 4 player gets a back shot


Control
- up arrow
- down arrow
- right arrow
- left arrow
