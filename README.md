Asteroids game in python featuring
-  auto shoot
-  player spaceshit with momentum
-  a scoring system (10 pts per hit, 5 more pts on small asteroids)
-  a stage system (move up stage every 200 pts)
-  a dynamic difficuly (asteroids spawn's frequency increases at every stage but so does the player fire rate)
-  at stage 4 player gets a back shot


Control
- up arrow      : accelerate forward
- down arrow    : decelerate and move slowly backward
- right arrow   : turn right
- left arrow    : turn left
- space         : pause the game
- escape        : quit game
